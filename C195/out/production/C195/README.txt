## Directions
Once you log in with one of the users mentioned above you will find yourself at the main screen. From here you will see
all of the existing appointments in the top table view under Appointments. You can click the Add Appointment button
add a new appointment. You can select an existing appointment and click the Modify Appointment button to make any changes.
You can select an existing appointment then click the Delete Appointment to delete an appointment.

On the bottom table view you will see all Customers. If you want to add a new customer you can click the Add Customer
button. If you want to change an existing customer, select which customer to modify and then click the Modify Customer button.
If you want to delete a customer, select the customer and then click the Delete Customer button.

Under the Controls section you can generate several reports. The View Customer Reports By Month will allow you
to view appointments for a particular customer and then to view them by Month or by Week. If you click the View Customer
Reports By Type this will generate a report to view the appointments by a particular customer and then organize them by
appointment type. If you click the View Schedule by Contact this will generate a report that will allow you to select
a particular contact and then view all appointments assigned to them. If you click the Generate Contact Email List this
will generate a report of all emails on file for the contacts.

The last two buttons will navigate you either back to the login prompt or exit the program entirely. Click the <b>Log Off</b>
button will return to the login screen where you can log in as a different user. Click <b>Exit Program</b> and you will close
the Schedule Manager all together.
